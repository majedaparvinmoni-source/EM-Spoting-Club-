# EM-Spoting-Club-
Assalamu Alaikum এখন হয়তোবা আমাদের ক্লাবে ভর্তি হতে ইচ্ছুক এমন একজন খেলোয়ার আমার এই প্রতিবেদনটি পড়ছে আসলে আমাদের এই ক্লাবটি প্রথম 2022 সালে গঠিত হয় আপনারা জানেন যে ক্লাবটি কে গঠিত করেছে ১৪ থেকে ১৫ বছরের দুজন বাচ্চা   তারা এর জন্য নিজেরা লোগো তৈরি করেছে নাম দিয়েছে তারা হচ্ছে মিনহাজ ও ইমতিয়াজ পরবর্তীতে ইনমতিয়াজ চলে গেলে মিনহাজ  এটি গড়ে তোল
